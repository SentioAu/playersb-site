PlayersB is tool-first. This section explains the minimum methodology behind our comparisons so you can trust what you see and understand the tradeoffs.

Core principles

  • Normalize before comparing: totals can mislead when minutes differ, so we use per-90 / per-minute metrics where appropriate.
  • Explainable outputs: if we compute a score (e.g., consistency), we show what goes into it.
  • No predictions: PlayersB does not publish “locks” or guaranteed outcomes.

Key concepts (short, practical)

Per-90 (why it exists)

Per-90 answers: “If this player played a full match, what would their rate look like?” It helps compare players with different minutes played.

Efficiency vs volume

Some players produce by volume (many shots/touches), others by efficiency (higher conversion). Comparisons should show both—otherwise you misread the role.

Consistency (simple definition)

Consistency isn’t “best player.” It’s “how stable is the output over games?” A basic version can use variability of recent outputs relative to the average. Lower variability → more consistent.
